1. Introduction to Java – Simple Notes (Exam Ready)
1. Java

Definition

Java is a high-level, object-oriented, platform-independent programming language developed by James Gosling at Sun Microsystems in 1995. It is used to develop web, desktop, mobile, and enterprise applications.

Example

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello Java");
    }
}

Output  Hello Java


2. History of Java
Definition

Java was developed by James Gosling and his team at Sun Microsystems in 1995.

Important Points
Initially called Oak.
Renamed to Java because "Oak" was already trademarked.
In 2010, Oracle Corporation acquired Sun Microsystems and now maintains Java.

5. JVM (Java Virtual Machine)
Definition

JVM is a virtual machine that executes Java bytecode and converts it into machine code.

Functions
Executes Java programs
Provides platform independence
Performs memory management
Handles garbage collection

Diagram
Java Program
      ↓
   Bytecode
      ↓
     JVM
      ↓
 Machine Code


6. JRE (Java Runtime Environment)

Definition

JRE provides the environment required to run Java applications.

Contains
JVM
Java Libraries
Supporting files
Formula
JRE = JVM + Libraries


7. JDK (Java Development Kit)
Definition

JDK is used to develop, compile, and run Java programs.

Contains
JRE
Compiler (javac)
Debugger
Development tools
Formula
JDK = JRE + Development Tools


8. Bytecode
Definition

Bytecode is the intermediate code generated by the Java compiler (javac).

Important Points
Stored in .class files.
Executed by the JVM.
Makes Java platform independent.
